Key Terms

Rendering Engine

Core that parses HTML/CSS/JS and paints pages (e.g., Blink, WebKit, Gecko).

Browser Wars

1990s competition—Netscape vs. Internet Explorer—shaped today’s market.

Sandboxing

Security isolation that limits what a web page can access on your system.

Phishing

Deceptive messages or sites that trick users into giving credentials or data.

HTTPS

Encrypted HTTP that protects data in transit and verifies site identity.

Progressive Web App (PWA)

Installable web apps with offline support and native-like features.

Agentic Browser

Assistant-style browser that can summarize, automate tasks, and personalize browsing.
